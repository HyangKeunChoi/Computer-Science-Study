
## 1980년대

+ 개인용 컴퓨터 시대
  - 1980년대 이전: 대형 컴퓨터를 여러 명이 접속해서 사용 (UNIX) - 시분할 시스템
  - 1980년대 ~: Personal Computer (PC)
  
+ 용어 이해: CLI와 GUI
  - CLI (Command Line Interface): 터미널 환경 (키보드, 검은 화면)
  - GUI (Graphical User Interface): GUI 환경(마우스)
